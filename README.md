# SOTA Tabular Modeling Library

Библиотека для обучения современных моделей на табличных данных с поддержкой:
- CatBoost
- LightGBM
- XGBoost
- LightAutoML
- RandomForest
- TabNet
- CEMLP (MLP с эмбеддингами для категориальных признаков)

## Основные возможности
- Единый API для обучения, валидации и предсказания
- Гибкая настройка моделей и пайплайна через YAML-конфиг
- Поддержка кросс-валидации, калибровки вероятностей, кастомных метрик
- Современные техники регуляризации и оптимизации (feature-wise dropout, dynamic embedding size, PReLU и др.)
- Легко расширяется новыми моделями и метриками

## Требования
- Python 3.12+
- PyTorch, scikit-learn, numpy, pandas, tqdm и др. (см. requirements.txt)

## Структура
- models/ — реализации моделей и их обёртки
- configs/ — примеры конфигов
- data/ — подготовленные датасеты

## Документация
- Все параметры моделей и пайплайна подробно описаны в my_config.yaml
- Для расширения добавьте новый класс модели в models/ и пропишите его в конфиге
